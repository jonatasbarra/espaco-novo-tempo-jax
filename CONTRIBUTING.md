# Guia de Contribuição - Espaço Novo Tempo Jax

Ficamos muito felizes com o seu interesse em contribuir para o website da nossa igreja! Para mantermos o projeto organizado, pedimos que siga os passos abaixo.

## Como Contribuir com Código ou Conteúdo

1. Faça um **Fork** do repositório.
2. Crie uma Branch para a sua alteração:
   \`\`\`bash
   git checkout -b minha-contribuicao
   \`\`\`
3. Faça as alterações necessárias no HTML, CSS ou JavaScript.
4. Faça o **Commit** das suas alterações com uma mensagem clara e descritiva:
   \`\`\`bash
   git commit -m 'Atualiza os horários de culto na página principal'
   \`\`\`
5. Faça o **Push** para a sua Branch:
   \`\`\`bash
   git push origin minha-contribuicao
   \`\`\`
6. Abra um **Pull Request** detalhando o que foi alterado.

## Reportar Problemas ou Sugerir Melhorias
Encontrou algum link quebrado, erro de digitação ou tem uma ideia para o site? 
Por favor, abra uma **Issue** no GitHub explicando o que aconteceu. Se for um erro visual, adicionar uma captura de ecrã (screenshot) ajuda bastante!
