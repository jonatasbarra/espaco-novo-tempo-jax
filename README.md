## 📖 Sobre o Projeto
Este website foi desenvolvido para ser o portal online da **Igreja Adventista Espaço Novo Tempo Jax**. O objetivo é partilhar informações sobre a nossa comunidade, horários de culto, eventos locais em Jacksonville, FL, e formas de contacto, proporcionando uma experiência acolhedora e acessível a todos os visitantes.

## 💻 Tecnologias Utilizadas
O projeto foi construído utilizando as tecnologias base da web, visando simplicidade e boa performance:
- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização e design responsivo.
- **JavaScript**: Interatividade e dinamismo da página.

## 🚀 Como Executar o Projeto

Como este é um projeto estático (HTML/CSS/JS), não é necessária uma instalação complexa de dependências.

1. Clone este repositório para a sua máquina local:
\`\`\`bash
git clone https://github.com/SEU-USUARIO/novo-tempo-jax.git
\`\`\`

2. Aceda à pasta do projeto:
\`\`\`bash
cd novo-tempo-jax
\`\`\`

3. Abra o ficheiro `index.html` diretamente no seu browser duplo clique, ou utilize uma extensão como o **Live Server** (no VS Code) para ter um ambiente de desenvolvimento com atualização automática.
